Tiny Charts

A curl interface to piety charts Access-Control-Allow-Origin: * included

Pie Charts

curl http://peity.herokuapp.com/ -d "type=pie" -d "data=1/5"

Line Charts

curl http://peity.herokuapp.com/ -d "type=line" -d "data=5,3,9,6,5,9,7,3,5,2"

Bar Charts

curl http://peity.herokuapp.com/ -d "type=bar" -d "data=5,3,9,6,5,9,7,3,5,2"
